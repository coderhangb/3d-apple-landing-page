# 3D Apple Landing Page

This project was built strictly for **educational and practice purposes**. The main goal was to explore the synergy between 3D rendering and modern web animations.

## 🚀 Learning Objectives
The primary focus of this repository is to practice:
- **Three.js**: Rendering 3D models, managing lighting, and materials.
- **GSAP (GreenSock Animation Platform)**: Creating high-performance, scroll-based animations using `ScrollTrigger`.
- **React Three Fiber**: Integrating 3D scenes into a React environment efficiently.

## 🛠 Tech Stack
- **Framework**: React.js
- **3D Library**: Three.js (via React Three Fiber & Drei)
- **Animation**: GSAP
- **Styling**: Tailwind CSS

## 📌 Note
This is a **practice-only** project. The design and assets are inspired by Apple’s product landing pages to study how they implement high-end visual storytelling on the web.
